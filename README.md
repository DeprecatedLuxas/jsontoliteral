# jsontoliteral

> A nodejs cli to convert a json file to typescript string literal

## Install

```bash
$ npm install jsontoliteral
```

## Usage
JSON file format
```json
{
  "literals": [
    {
      "literal": "Dog"
    },
    {
      "literal": "Cat"
    },
    {
      "literal": "Cow"
    }
  ]
}
```


```bash
$ jsontoliteral animals.json
```
